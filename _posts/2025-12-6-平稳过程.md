## 1. 平稳过程概述

### 1.1 核心定位

平稳过程是一类统计特性**不随时间推移而变化的随机过程**，广泛应用于通讯理论、自动控制和预报问题等领域。

### 1.2 与马尔科夫过程的区别

马尔科夫过程的未来状态仅依赖于**当前状态**，而平稳过程的未来状态会受到**过去所有情况的显著影响**，无法归入马尔科夫过程范畴。

### 1.3 研究意义

现实中很多系统（如稳定运行的电路、正常传播的信号）的随机过程可近似为平稳过程，其理论能简化实际问题的分析和求解。

## 2. 平稳过程的定义与分类

### 2.1 强平稳过程（严平稳过程）

#### 2.1.1 定义

对于随机过程$\{X(t), t \in T\}$，若对任意正整数$n$、任意时刻$t_1, \cdots, t_n$和任意时间平移$h$（满足$t_1+h, \cdots, t_n+h \in T$），$n$维随机向量$(X(t_1), \cdots, X(t_n))^T$与$(X(t_1+h), \cdots, X(t_n+h))^T$的分布完全相同，即：
$$F_n(x_1, \cdots, x_n, t_1, \cdots, t_n) = F_n(x_1, \cdots, x_n, t_1+h, \cdots, t_n+h)$$
则称该过程为强平稳过程。

#### 2.1.2 关键注记

强平稳的核心是“有限维分布不随时间平移变化”，且高阶平稳一定是低阶平稳，但低阶平稳不一定是高阶平稳。

**什么是低阶/高阶平稳？**

低阶 / 高阶平稳针对强平稳过程，以 “满足平稳性的有限维分布的维度 n” 为划分标准：n 的**上限**越小，平稳阶数越低；n 的**上限**越大，平稳阶数越高。

### 2.2 弱平稳过程（宽平稳过程）

#### 2.2.1 定义条件

满足以下三个条件的随机过程称为弱平稳过程（默认讨论的平稳过程均指此类）：

1. 是二阶矩过程（即对所有$t \in T$，$E[X^2(t)] < \infty$）；
2. 均值函数为常数：$E[X(t)] = \mu$（与$t$无关）；
3. 相关函数仅与时间差有关：$R(t_1, t_2) = E[X(t_1)X(t_2)] = R(t_2 - t_1) = R(\tau)$（$\tau = t_2 - t_1$）。

**如何判断一个随机过程是否为弱平稳过程？**

1. 计算$E[X^2(t)]$，验证二阶矩是否存在。
2. 计算均值函数$E[X(t)]$，验证是否为常数。
3. 计算自相关函数$R(t_1, t_2)$，验证是否只和时间差$\tau = t_2 - t_1$有关。

#### 2.2.2 条件简化

可省去“二阶矩过程”条件，改为验证$R(t) = E[X^2(t)] < \infty$即可。

### 2.3 强平稳与弱平稳的关系

- 强平稳不一定是弱平稳：强平稳过程可能不存在二阶矩；
- 弱平稳不一定是强平稳：前二阶矩无法完全确定有限维分布；
- 特殊等价情形：若平稳过程是正态过程，则强平稳与弱平稳等价（正态分布由均值和协方差完全确定）。

**为什么强平稳过程可能不存在二阶矩？**

强平稳的定义仅约束 “有限维分布不随时间平移变化”，未对 “二阶矩（$E [X²(t)]$）有限” 做任何要求，为两个独立属性。

**为什么前二阶矩无法完全确定有限维分布？举一个三阶矩不是有限维的情况。**

前二阶矩（均值、协方差 / 相关函数）仅能描述随机过程的 “位置”“离散程度” 和 “线性关联”，无法捕捉分布的高阶特征（如偏度、峰度）和整体形状；而有限维分布包含了随机变量取值的全部概率信息，因此仅前二阶矩不能完全确定有限维分布。

### 2.4 典型例子

#### 2.4.1 白噪声序列

随机序列$\{X(n), n = 0, \pm1, \cdots\}$满足：

$$
E[X(n)] = 0, \quad E[X(n)X(m)] = \begin{cases}0, & n \neq m \\ \sigma^2, & n = m\end{cases}
$$

是互不相关的平稳序列，正态分布的白噪声称为高斯白噪声序列。

#### 2.4.2 三角多项式过程

设$A_1, \cdots, A_N$和$B_1, \cdots, B_N$是均值为 0、互不相关的随机变量（$E[A_nA_m] = E[B_nB_m] = \sigma^2\delta_{nm}$，$E[A_nB_m] = 0$），$\omega_1, \cdots, \omega_N$为正实数，定义：

$$
X(t) = \sum_{k=1}^N (A_k\cos\omega_kt + B_k\sin\omega_kt)
$$

其均值$E[X(t)] = 0$，相关函数$R(\tau) = \sigma^2\sum_{k=1}^N \cos\omega_k\tau$，是平稳过程（圆频率$\omega_k$相同）。

#### 2.4.3 随机相位周期过程

设$S(t)$是周期为$T$的有界实函数，$\Theta$在$(0, T)$上均匀分布，定义$X(t) = S(t + \Theta)$。其均值为$\frac{1}{T}\int_0^T S(\theta)d\theta$（常数），相关函数仅与时间差有关，是平稳过程。

## 3. 平稳过程相关函数的性质

### 3.1 单个宽平稳过程的相关函数性质

设$R(\tau) = E[X(t)X(t+\tau)]$为自相关函数，$C(\tau) = R(\tau) - \mu^2$为协方差函数。

1. 非负性：$R(0) = E[X^2(t)] = \psi_X^2 \geq 0$（$R(0)$也是过程的“平均功率”）；
2. 对称性：$R(-\tau) = R(\tau)$（仅需研究$\tau \geq 0$的情况）；
3. 有界性：$\|R(\tau)\| \leq R(0)$，$\|C(\tau)\| \leq C(0) = \sigma_X^2$（在$\tau=0$处取得最大值）；
4. 周期性：若$X(t)$是周期为$T_0$的周期平稳过程，则$R(\tau)$也是周期为$T_0$的函数。

### 3.2 两个联合平稳过程的互相关函数性质

设$\{X(t)\}$和$\{Y(t)\}$联合平稳，$R_{XY}(\tau) = E[X(t)Y(t+\tau)]$为互相关函数：

1. 对称性：$R_{XY}(\tau) = R_{YX}(-\tau)$（与$X$、$Y$的顺序相关）；
2. 有界性：$\|R_{XY}(\tau)\|^2 \leq R_X(0)R_Y(0)$，$\|C_{XY}(\tau)\|^2 \leq C_X(0)C_Y(0)$；
3. 复值情形：若为复值平稳过程，则$R_{XY}(\tau) = \overline{R_{YX}(-\tau)}$（$\overline{\cdot}$表示共轭）。

## 4. 均方分析

### 4.1 基础概念

二阶矩过程：对所有$t \in T$，$E[X^2(t)] < \infty$的随机过程（均方分析的研究对象）。

### 4.2 均方极限

#### 4.2.1 定义

设$\{X_n\}$是二阶矩随机序列，$X$是二阶矩随机变量，若$\lim_{n \to \infty} E[\|X_n - X\|^2] = 0$，则称$\{X_n\}$均方收敛于$X$，记为$X_n \stackrel{m.s.}{\to} X$。

#### 4.2.2 核心性质

1. 唯一性：若$X_n \stackrel{m.s.}{\to} X$且$X_n \stackrel{m.s.}{\to} Y$，则$P(X = Y) = 1$（几乎处处相等）；
2. 线性运算：若$X_n \stackrel{m.s.}{\to} X$，$Y_n \stackrel{m.s.}{\to} Y$，$a, b$为常数，$c_n \to c$（数列收敛），则$aX_n + bY_n \stackrel{m.s.}{\to} aX + bY$，$c_nX_n \stackrel{m.s.}{\to} cX$；
3. 期望可交换：$\lim_{n \to \infty} E[X_n] = E[\lim_{n \to \infty} X_n] = E[X]$；
4. 乘积期望可交换：$\lim_{m,n \to \infty} E[X_nY_m] = E[XY]$。

#### 4.2.3 收敛准则（等价条件）

1. $X_n \stackrel{m.s.}{\to} X$；
2. $\lim_{m,n \to \infty} E[X_mX_n] = E[\|X\|^2]$；
3. $\lim_{m,n \to \infty} E[\|X_n - X_m\|^2] = 0$（$\{X_n\}$是均方柯西序列）。

#### 4.2.4 特殊情形

正态序列的均方极限仍是正态随机变量，且均值和方差分别为原序列均值和方差的极限。

### 4.3 均方连续

#### 4.3.1 定义

设$\{X(t), t \in T\}$是二阶矩过程，若对$t_0 \in T$，$\lim_{h \to 0} X(t_0 + h) \stackrel{m.s.}{=} X(t_0)$，则称$X(t)$在$t_0$处均方连续；若在$T$上所有点均连续，则称过程在$T$上均方连续。

#### 4.3.2 充要条件

$X(t)$在$t_0$处均方连续的充要条件是其相关函数$R(t_1, t_2)$在$(t_0, t_0)$处连续。

#### 4.3.3 推论

若$R(t_1, t_2)$在对角线$\{(t,t) \| t \in T\}$上连续，则$R(t_1, t_2)$在$T \times T$上全域连续，且$X(t)$在$T$上均方连续。

### 4.4 均方可导

#### 4.4.1 定义

设$\{X(t), t \in T\}$是二阶矩过程，若对$t_0 \in T$，均方极限$\lim_{h \to 0} \frac{X(t_0 + h) - X(t_0)}{h}$存在，则称$X(t)$在$t_0$处均方可导，该极限记为$X'(t_0)$（均方导数）；若在$T$上所有点均可导，则称过程在$T$上均方可导。

#### 4.4.2 充要条件

$X(t)$在$T$上均方可导的充要条件是其相关函数$R(t_1, t_2)$在对角线$\{(t,t) \| t \in T\}$上的广义二阶导数存在。

#### 4.4.3 核心性质

1. 均方可导必均方连续，但均方连续不一定均方可导；
2. 导数的数字特征：$E[X'(t)] = \mu_X'(t)$，$R_{X'X}(t_1, t_2) = \frac{\partial R(t_1, t_2)}{\partial t_1}$，$R_{X'X'}(t_1, t_2) = \frac{\partial^2 R(t_1, t_2)}{\partial t_1\partial t_2}$；
3. 线性运算：$(aX(t) + bY(t))' = aX'(t) + bY'(t)$（$a, b$为常数）；
4. 乘积法则：若$f(t)$是普通可微函数，$X(t)$均方可导，则$\frac{d}{dt}[f(t)X(t)] = f'(t)X(t) + f(t)X'(t)$；
5. 正态过程的导数：正态过程的均方导数仍是正态过程。

### 4.5 均方可积

#### 4.5.1 定义

设$\{X(t), t \in [a,b]\}$是二阶矩过程，对$[a,b]$的任意分割$a = t_0 < t_1 < \cdots < t_n = b$，取$t_k' \in [t_{k-1}, t_k]$，若均方极限

$$
\lim_{\Delta_n \to 0} \sum_{k=1}^n X(t_k')(t_k - t_{k-1}), \quad \Delta_n = \max_{1 \leq k \leq n} (t_k - t_{k-1})
$$

存在且与$t_k'$的取法无关，则称$X(t)$在$[a,b]$上均方可积，记为$\int_a^b X(t)dt$。

$\Delta _n$是所有分段中长度最长的分段长度，和当前分段长度无关。

#### 4.5.2 充要条件

$X(t)$在$[a,b]$上均方可积的充分必要条件是其相关函数$R(s,t)$在$[a,b] \times [a,b]$上可积。

#### 4.5.3 核心性质

1. 线性运算：$\int_a^b [aX(t) + bY(t)]dt = a\int_a^b X(t)dt + b\int_a^b Y(t)dt$；
2. 期望与积分可交换：$E\left[\int_a^b X(t)dt\right] = \int_a^b E[X(t)]dt$；
3. 乘积积分性质：$E\left[\int_a^b X(t)dt \cdot \int_c^d Y(t)dt\right] = \int_a^b \int_c^d R_{XY}(s,t)dsdt$；
4. 不定积分：设$Y(t) = \int_a^t X(\tau)d\tau$，则$Y(t)$在$[a,b]$上均方连续且均方可导，且$Y'(t) = X(t)$（均方意义下）。

## 5. 遍历性（各态历经性）

### 5.1 核心思想

平稳过程的统计平均（如均值、相关函数）可通过单个样本函数的时间平均近似代替，无需大量重复观测。

### 5.2 定义

设$\{X(t), t \in \mathbb{R}\}$是宽平稳过程：

1. 均值遍历性：若$\lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T X(s)ds \stackrel{a.e.}{=} \mu_X = E[X(s)]$（几乎处处收敛），则称均值具有遍历性；
2. 相关函数遍历性：若$\lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T X(s)X(s+\tau)ds \stackrel{a.e.}{=} R(\tau)$，则称相关函数具有遍历性；
3. 遍历过程：均值和相关函数均具有遍历性的平稳过程。

### 5.3 遍历性定理（充要条件）

#### 5.3.1 均值遍历性

平稳过程$\{X(t)\}$的均值具有遍历性的充要条件是：

$$
\lim_{T \to \infty} \frac{1}{T} \int_0^{2T} \left(1 - \frac{\tau}{2T}\right) C_X(\tau)d\tau = 0
$$

其中

$$
C_X(\tau) = R_X(\tau) - \mu_X^2
$$

是协方差函数。

**必要条件证明：**

由[定义](#5.2 定义)可设时间平均为：

$$
\eta = \lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T X(s)ds
$$

那么时间平均的期望为：

$$
\begin{align}
E(\eta) &= E[\lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T X(s)ds] \\
&= \lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T E[X(s)]ds
\end{align}
$$

根据统计平均：

$$
\mu_X = E[X(s)] = \mu (常数)
$$

可得：

$$
E(\eta) = \lim_{T \to \infty} \frac{1}{2T} \int_{-T}^T \mu \ ds = \mu(常数)
$$

【**时间平均收敛到平均统计**】的充要条件是$\eta$的方差$D\eta = 0$。

方差的定义是 $D\eta = E[\eta^2] - (E\eta)^2$，因此需要先计算 $E[\eta^2]$：

由于 $\eta$ 是积分的极限，利用**极限与期望的交换性**（平稳过程满足）：

$$
E[\eta^2] = \lim_{T \to \infty} E\left[ \left( \frac{1}{2T} \int_{-T}^{T} X(s)ds \right) \left( \frac{1}{2T} \int_{-T}^{T} X(t)dt \right) \right]
$$

展开后利用平稳过程的自相关性质 $E[X(s)X(t)] = R_X(s-t)$（自相关仅与时间差有关），得：

$$
E[\eta^2] = \lim_{T \to \infty} \frac{1}{4T^2} \int_{-T}^{T} \int_{-T}^{T} R_X(t - s) ds dt
$$

因此方差 $D\eta$ 为：

$$
D\eta = \lim_{T \to \infty} \frac{1}{4T^2} \int_{-T}^{T} \int_{-T}^{T} R_X(t - s) ds dt - \mu_X^2
$$

对二重积分 $\int_{-T}^{T} \int_{-T}^{T} R_X(t - s) ds dt$ 做**变量替换**：令 $\tau = t - s$（时间差），分析 $s,t$ 的范围（$s,t \in [-T,T]$）：

- 当 $\tau \in [-2T, 2T]$ 时，$s$ 的取值区间长度为 $2T - \|\tau\|$（可通过平面区域截距分析：$s-t$ 平面的正方形区域中，沿 $\tau=t-s$ 直线的截距长度为 $2T - \|\tau\|$）。

结合自相关函数的**偶性**（$R_X(\tau) = R_X(-\tau)$），二重积分可化简为：

$$
\int_{-T}^{T} \int_{-T}^{T} R_X(t - s) ds dt = \int_{-2T}^{2T} R_X(\tau) \cdot (2T - \|\tau\|) d\tau = 2 \int_{0}^{2T} R_X(\tau) \cdot (2T - \tau) d\tau
$$

将二重积分的结果代入 $D\eta$：

$$
D\eta = \lim_{T \to \infty} \frac{1}{4T^2} \cdot 2 \int_{0}^{2T} R_X(\tau)(2T - \tau) d\tau - \mu_X^2
$$

化简系数与积分项：

$$
D\eta = \lim_{T \to \infty} \frac{1}{T} \int_{0}^{2T} \left(1 - \frac{\tau}{2T}\right) R_X(\tau) d\tau - \mu_X^2
$$

再利用 $\int_{0}^{2T} \left(1 - \frac{\tau}{2T}\right) d\tau = T$（直接积分可得），将 $\mu_X^2$ 凑成积分形式：

$$
\mu_X^2 = \frac{1}{T} \int_{0}^{2T} \left(1 - \frac{\tau}{2T}\right) \mu_X^2 d\tau
$$

因此 $D\eta$ 可改写为：

$$
D\eta = \lim_{T \to \infty} \frac{1}{T} \int_{0}^{2T} \left(1 - \frac{\tau}{2T}\right) \left(R_X(\tau) - \mu_X^2\right) d\tau
$$

由于均值遍历性的充要条件是 $D\eta = 0$，因此等价于：

$$
\lim_{T \to \infty} \frac{1}{T} \int_{0}^{2T} \left(1 - \frac{\tau}{2T}\right) \left(R_X(\tau) - \mu_X^2\right) d\tau = 0
$$

#### 5.3.2 相关函数遍历性

平稳过程$\{X(t)\}$的自相关函数$R_X(\tau)$具有遍历性的充要条件是：

$$
\lim_{T \to \infty} \frac{1}{T} \int_0^{2T} \left(1 - \frac{\tau_1}{2T}\right) \left(B(\tau_1) - R_X^2(\tau)\right)d\tau_1 = 0
$$

其中

$$
B(\tau_1) = E[X(t+\tau+\tau_1)X(t+\tau_1)X(t+\tau)X(t)]
$$

是四阶矩相关量。

### 5.4 实际近似估计

由于无法观测无限长时间（$T \rightarrow \infty$），通常取有限区间$[0,T]$的样本$x(t)$，近似估计：

1. 均值估计：$\hat{\mu}_X = \frac{1}{T} \int_0^T x(t)dt$；
2. 相关函数估计：$\hat{R}_X(\tau) = \frac{1}{T - \tau} \int_0^{T - \tau} x(t)x(t+\tau)dt$（$0 \leq \tau < T$）。

**如何分析一个过程的遍历性？或者说如何判断一个过程是否具有遍历性？**

1. 判断前提：先确认过程是 “宽平稳过程”，判断标准见[宽平稳过程的定义条件](#2.2.1 定义条件)。
2. 判断 “均值遍历性”（核心且常用）

   - [定义](#5.2 定义)（根本判断）
   - [理论充要条件](#5.3.1 均值遍历性)（精准判断，积分过于复杂）
   - 实际简化验证方法（工程常用）：验证$\int_{-\infty}^{+\infty} \|C_X(\tau)\| d\tau < \infty$，即$C_X(\tau)$收敛

3. 判断 “相关函数遍历性”
   - [定义](#5.2 定义)（根本判断）
   - [理论充要条件](#5.3.1 均值遍历性)（精准判断，积分过于复杂）
   - 关键简化：正态平稳过程若$\{X(t)\}$是正态平稳过程，四阶矩可由二阶矩表示（正态过程高阶矩由二阶矩完全确定）：$B(\tau_1) = R_X^2(\tau) + R_X^2(\tau_1) + 2R_X(\tau+\tau_1)R_X(\tau-\tau_1)$代入后可简化积分计算，无需额外求四阶矩。

## 基本符号解释

### 随机过程与变量

| 符号/表达式                     | 含义说明                                                                  |
| ------------------------------- | ------------------------------------------------------------------------- |
| $\{X(t), t \in T\}$             | 随机过程，$t$ 为时间参数，$T$ 为时间参数集（如 $\mathbb{R}$、$[a,b]$ 等） |
| $\{X(n), n = 0, \pm1, \cdots\}$ | 随机序列（离散时间随机过程）                                              |
| $X, Y, X_n$                     | 随机变量（$X_n$ 为随机序列的第 $n$ 项）                                   |
| $\Theta$                        | 随机相位（常用于随机相位周期过程）                                        |
| $A_k, B_k$                      | 互不相关的随机振幅（常用于三角多项式过程）                                |

### 统计量符号

| 符号/表达式               | 含义说明                                                                           |
| ------------------------- | ---------------------------------------------------------------------------------- |
| $\mu_X = E[X(t)]$         | 随机过程 $\{X(t)\}$ 的均值（弱平稳过程中为常数）                                   |
| $E[X(t)]$                 | 随机变量 $X(t)$ 的数学期望                                                         |
| $E[X^2(t)]$               | 随机变量 $X(t)$ 的二阶矩（弱平稳过程中 $E[X^2(t)] = R_X(0)$）                      |
| $R_X(\tau) = R(t_1, t_2)$ | 自相关函数，$\tau = t_2 - t_1$ 为时间差，$R_X(\tau) = E[X(t)X(t+\tau)]$            |
| $R_{XY}(\tau)$            | 互相关函数，$R_{XY}(\tau) = E[X(t)Y(t+\tau)]$（$\{X(t)\}$ 与 $\{Y(t)\}$ 联合平稳） |
| $C_X(\tau)$               | 自协方差函数，$C_X(\tau) = R_X(\tau) - \mu_X^2$                                    |
| $C_{XY}(\tau)$            | 互协方差函数，$C_{XY}(\tau) = R_{XY}(\tau) - \mu_X\mu_Y$                           |
| $\sigma_X^2$              | 方差，$\sigma_X^2 = C_X(0) = E[(X(t) - \mu_X)^2]$                                  |
| $\psi_X^2$                | 二阶原点矩，$\psi_X^2 = R_X(0) = E[X^2(t)]$                                        |
| $B(\tau_1)$               | 四阶矩相关量，$B(\tau_1) = E[X(t+\tau+\tau_1)X(t+\tau_1)X(t+\tau)X(t)]$            |

### 极限与收敛符号

| 符号/表达式                   | 含义说明                                                        |
| ----------------------------- | --------------------------------------------------------------- | ------- | -------- |
| $\lim_{n \to \infty} X_n = X$ | 普通极限（数列或随机变量序列的极限）                            |
| $X_n \stackrel{m.s.}{\to} X$  | 均方收敛，即 $\lim\_{n \to \infty} E[                           \| X_n - X \| ^2] = 0$ |
| $X_n \stackrel{P}{\to} X$     | 依概率收敛                                                      |
| $X_n \stackrel{a.e.}{\to} X$  | 几乎处处收敛                                                    |
| $X_n \stackrel{d}{\to} X$     | 依分布收敛                                                      |
| $\lim_{\Delta_n \to 0}$       | 分割区间的最大步长 $\Delta_n$ 趋于 0 的极限（用于均方积分定义） |

### 运算与特殊符号

| 符号/表达式       | 含义说明                                                                       |
| ----------------- | ------------------------------------------------------------------------------ |
| $\int_a^b X(t)dt$ | 均方积分（随机过程 $X(t)$ 在 $[a,b]$ 上的均方积分）                            |
| $X'(t)$           | 均方导数（随机过程 $X(t)$ 的均方导数）                                         |
| $\overline{Z}$    | 复数值 $Z$ 的共轭（用于复值平稳过程的互相关函数）                              |
| $\delta_{nm}$     | 克罗内克 delta 符号，$\delta_{nm} = 1$（$n=m$），$\delta_{nm}=0$（$n \neq m$） |
| $\omega_k$        | 圆频率（常用于三角多项式过程、随机相位正弦波）                                 |
| $T$               | 周期（用于周期平稳过程、随机相位周期过程）                                     |
| $T_0$             | 平稳过程的周期                                                                 |
| $\tau$            | 时间差（用于相关函数、协方差函数）                                             |
